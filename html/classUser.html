User ADT class and variables description *

  • d_id -> Unique identifier (ID) for each user (client). * d_name -> User's Full name. * d_city -> User's city. * d_age -> User's age. * d_money -> User's money * d_time_spent -> User's time spent inside the casino. * d_prizes_won -> User's total amount of prizes won. * d_bets -> User's bet within a game round. * d_debt -> User's debt to the casino. * count_plays_no_money -> Counter for number of times user tried to play without money. * playing -> Flag to control whether user is playing. \NOT USED *
