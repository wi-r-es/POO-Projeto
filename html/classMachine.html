Detailed Description

Machine ADT class description *

  • ID -> TYPE_TBD of the Unique identifier (ID) for each machine. * state -> Current state of the machine. * type -> Type of game the machine offers. * failureProbability -> Generic LL for all products ("database"). * temperature -> Current temperature of the machine. * posX,posY -> Position of the machine in the casino (X, Y). * winProbability -> Probability that a user will win at this machine. * failures -> Number of times the machine has failed. * betAmount -> the bet amount of the round. * usage -> Number of times the machine was used. * coolingtime -> Time point object to simulate the machine getting fixed. *
